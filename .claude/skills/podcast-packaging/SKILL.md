# Skill: Podcast Packaging

## Purpose
Create publish-ready podcast assets while recording or immediately after.

## Outputs
- Episode title variants (3)
- Episode description (1 long, 1 short)
- Chapter markers (timestamped)
- Quote bank (timestamped)
- Promo pack: X thread + LinkedIn + IG caption

## Procedure
1. Detect topic shifts and produce chapter markers.
2. Extract memorable quotes (avoid private details).
3. Draft metadata with conservative claims; mark items that need verification.
