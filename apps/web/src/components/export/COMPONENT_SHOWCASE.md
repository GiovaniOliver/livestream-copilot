## Color Scheme

### Platform Colors
- **X (Twitter)**: #1DA1F2 (Light Blue)
- **LinkedIn**: #0A66C2 (Blue)
- **Instagram**: #E4405F (Pink/Red)
- **TikTok**: #000000 (Black)
- **YouTube**: #FF0000 (Red)

### UI States
- **Selected/Active**: Teal (#00D4C7)
- **Success**: Green (#2EE59D)
- **Warning**: Amber (#FBBF24)
- **Error**: Red/Error Color
- **Purple Accent**: #8B5CF6

### Interactions
- **Hover**: Lighter background
- **Focus**: Ring with teal color
- **Active**: Slight scale down
- **Disabled**: 50% opacity

## Responsive Behavior

### Desktop (> 1024px)
- Modal: 4xl width (896px)
- Platform Grid: 5 columns
- Full labels and descriptions

### Tablet (768px - 1024px)
- Modal: 2xl width (672px)
- Platform Grid: 4 columns
- Abbreviated labels

### Mobile (< 768px)
- Modal: Full width with margin
- Platform Grid: 3 columns
- Icon-only platforms
- Stacked form fields

## Accessibility Features

### Keyboard Navigation
- Tab through all interactive elements
- Enter/Space to select platforms
- Escape to close modal
- Arrow keys for grid navigation

### Screen Reader
- Descriptive ARIA labels
- State announcements
- Progress updates
- Error messages

### Visual
- High contrast colors
- Focus indicators
- Loading states
- Error states

## Animation Timings

- Modal open/close: 300ms ease
- Button hover: 200ms ease
- Copy success: 2000ms display
- Download progress: Real-time
- Platform selection: 200ms ease
- Success states: Fade after 2s
