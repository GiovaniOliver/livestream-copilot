What FluxBoard is

FluxBoard is a hackathon prototype for a workflow-first agentic system that turns live sessions into structured, shareable assets. Users pick a session workflow (Streamer, Podcast, Writers Room, Brainstorm, Debate) and a capture mode (Audio, Video, or Audio+Video). The system then generates real-time outputs—posts, clips, chapters, claims, evidence cards, beats—rendered on dashboards that match the activity.

Architecture (high level)

Capture layer

OBS companion service for replay-buffer clips and screenshots; mobile app for session control and review.

Event stream

A single event schema (moments, artifacts, outputs) that every workflow dashboard can render.

Agents

Role-based agents generate workflow outputs (social drafts, script inserts, claim maps, packaging).

Observability with Comet Opik

FluxBoard is designed to be traceable end-to-end. In the scaffold repo, the desktop companion includes an Opik instrumentation module that can log traces and spans for clip capture and output generation.

Environment

Set OPIK_API_KEY, OPIK_WORKSPACE_NAME, and OPIK_PROJECT_NAME. For Opik Cloud, set OPIK_URL_OVERRIDE to the hosted API endpoint.

Brand

Theme: dark base with a purple→teal→mint gradient. Logo mark is a comet trail—nodding to “Comet” observability—and a “flow” motif for real-time agent outputs.
