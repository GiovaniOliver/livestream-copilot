Turn live sessions into publishable assets—while they happen.

FluxBoard is a workflow-first copilot for streamers, podcast teams, writers rooms, brainstorms, and debates. Pick a workflow, capture from OBS or mobile, and review agent outputs on dashboards that match the moment.

What you get in real time

Clip queue

Start/end clip intent markers, replay-buffer saves, and titles/hooks ready for Shorts/Reels.

Post drafts

Platform-ready captions and threads generated from the last segment of the session.

Attribution

Idea ledger and quote bank with timestamps and speaker tags (when available).

Debate cards

Claims map, verification queue, and evidence-card drafts for structured debates.

Podcast packaging

Chapters, promos, show notes, and a highlight list for post-production.

Operator controls

Approve, edit, copy, and share outputs instantly—no waiting for post-processing.

Workflow-first features

Instead of a generic “transcript viewer,” FluxBoard gives each session type its own visual dashboard—Producer Desk, Script Studio, Mind Map, or Claims & Evidence Board. Under the hood, every workflow uses the same event model so agents can generate outputs consistently.

Capture

OBS or Mobile

Use OBS for replay-buffer clips and source screenshots, or run a mobile-only session when you’re away from your desktop.

Modes

Audio / Video / A+V

Pick what you want to capture for each session. Audio-only is fast and transcript-first; video enables clips and thumbnails.

Actions

Copy, approve, share

Every output card is built for speed: copy to clipboard, quick edits, and a queue for what to publish next.

Workflows

Select a workflow at session start to activate the right agents and the right dashboard.

Live Streamer

Producer Desk: clip queue, post drafts by platform, and a moment rail for fast publishing.

Podcast

Podcast Console: chapters, quote bank, promos, and a highlights list with timestamps.

Writers Room

Script Studio: screenplay-style view, beat board, and attribution ledger for ideas and decisions.

Brainstorm

Mind Map Room: clusters, sticky nodes, and an action funnel to convert ideas into tasks.

Debate

Claims & Evidence Board: claims graph, verification queue, and evidence cards (with sourcing).

Agent observability (Comet Opik)

FluxBoard is instrumented for trace-level visibility so you can debug agent behavior and measure improvements. Opik supports agent observability via its TypeScript SDK, OpenTelemetry, and REST APIs.

Trace every step

Log traces/spans for session start, clip capture, output generation, and validation.

Annotate and iterate

Use the trace history to find bottlenecks, prompt regressions, and tool failures—fast.

Hackathon-friendly

Turn observability on with env vars. Keep it off for a local-only demo.

Configure OPIK_API_KEY, OPIK_WORKSPACE_NAME, and optional OPIK_URL_OVERRIDE to log traces to Opik.
