# Workflows

Users select a workflow at session start. Each workflow activates a different agent set and a different dashboard skin.

All workflows support `captureMode`:

- **audio**: transcript-first; no clips/screenshots required.
- **video**: visuals-first; clips/screenshots enabled; transcript optional.
- **av**: full pipeline (transcript + clips + screenshots).

---

## 1) Live Streamer
**Goal:** Generate social drafts and a clip queue during a live stream.

**Primary outputs:**
- Social post drafts (X, LinkedIn, IG, YT)
- Clip candidates (timestamps, titles/hooks, thumbnails)
- “Moment” markers (hype, Q&A, sponsor)

**Agent roles:**
- Live Social Producer
- Clip & Hook Strategist
- Brand/Compliance Gate

---

## 2) Writers Room
**Goal:** Convert a group writers session into structured script artifacts with attribution.

**Primary outputs:**
- Beat board (story beats)
- Script insert suggestions (formatted)
- Attribution ledger (who proposed what)
- Open loops and assignments

**Agent roles:**
- Brainstorm Scribe
- Story Editor
- Credits & Attribution Keeper

---

## 3) Brainstorming Session
**Goal:** Capture raw ideas, cluster them, and turn them into action plans.

**Primary outputs:**
- Idea nodes (tagged)
- Clusters/themes
- Shortlist & priority list
- Action items (owner, due)

**Agent roles:**
- Idea Capture
- Clustering/Facilitator
- Action Item Extractor

---

## 4) Debate
**Goal:** Maintain a claims/rebuttals map and a fact-check queue with evidence cards.

**Primary outputs:**
- Claim cards (IDs, speaker, timestamp)
- Rebuttal links (claim graph)
- Evidence cards (neutral summary + sources)
- Moderator prompts

**Agent roles:**
- Claims Extractor
- Fact-check Triage
- Evidence Card Writer
- Debate Moderator

---

## 5) Podcast
**Goal:** Produce publish-ready packaging while recording.

**Primary outputs:**
- Chapter markers (timestamped)
- Quote bank (timestamped)
- Episode title + description variants
- Promo copy (posts, newsletter blurb)
- Clip queue (shorts/reels)
- Publishing checklist

**Agent roles:**
- Chapter Marker
- Quote Curator
- Promo Copywriter
- Packaging/Export Agent
